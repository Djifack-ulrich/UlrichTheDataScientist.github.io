Web Scraping on Linkedin and Dasboard with Streamlit:
Event Search

One of the ways to grow professionally and connect with a network is by participating in events. In this project, we propose a solution that allows users to easily find events on LinkedIn. The user simply enters a keyword, and a web scraping process is performed in the background. The results are then displayed on a dashboard, which shows the number of events available. Users can also download a dataset containing event details such as the event name, summary, number of participants, date, and event link. Additionally, the dashboard presents the top five events with the highest number of participants.
